# MWRD

MWRD is a controlled B2B procurement marketplace with Client, Supplier, and Admin portals.

## Stack

- React + TypeScript + Vite
- Convex
- Better Auth with Convex integration
- Tailwind CSS + shadcn/ui
- React Router
- React Hook Form + Zod
- TanStack Table
- i18next
- PostHog
- Vitest + React Testing Library + Playwright
- pnpm

## Commands

```bash
pnpm install
pnpm dev
pnpm convex dev
pnpm typecheck
pnpm lint
pnpm test
pnpm test:e2e
pnpm build
pnpm seed
```
